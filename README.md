# roles.es

roles.es website

## Overview

This repository contains the source code for the roles.es website. It's a static website built with HTML and JavaScript, designed to showcase information about roles and related content.

## Technologies Used

- **HTML** (96.8%) - Structure and markup of the website
- **JavaScript** (3.2%) - Client-side interactivity and functionality

## Project Structure

The repository is organized as a simple static website with HTML pages and JavaScript files for enhanced user experience and interactive features.

## Getting Started

To view or contribute to this project, clone the repository and open the HTML files in your browser.

```bash
git clone https://github.com/BernatGames/roles.es.git
```

## How to update website content

Clean push:
```
git subtree push --prefix soveliorsage origin gh-pages
```

Forced push:
```
git push origin `git subtree split --prefix soveliorsage`:gh-pages --force
```
